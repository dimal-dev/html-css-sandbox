Exercise 2-12

üßë‚Äçüíª
Description

Create the 'images' folder (if it doesn't exist) beside your 2-12.html file for this exercise.

Create the 'jobs' folder inside the 'images' folder.

Save the images from the previous exercise into the 'jobs' folder.

To save an image, right-click on the image and then select 'Save Image As...'

You will end up with the following file structure.

Then just like in the previous exercise create a webpage with clickable images to the job posting websites.

But this time use relative URLs to the images you've saved into the images/jobs folder.

So for example instead of 

<img src="https://dimal-dev.github.io/html-css/exercise/assets/lesson-2-images/linkedin.png">
you'll have 

<img src="images/jobs/linkedin.png">
